Itemized Line Item Editor

Repeating-row editor inside EstimateChangeOrderDetails when creationType = Itemized. Add Item opens a searchable combobox (Labor trade levels, Materials, Equipment, Custom). Per-line markup %; totals roll to IE-3. Mirrors SimpleInvoice.tsx row patterns (quantity, unit cost, extension).

Estimate: 4–6 days Parent: UIS-IE

1. Business Case ▼

Estimators need the same mental model as field T&M capture: pick catalog rows or type free-form descriptions, see unit cost populate, adjust quantity and markup.

Combination of dropdown picks and manual dollar entry on the same grid. — Clement / product direction (April 2026)

2. Scope ▼

In Scope

  • IN EstimateLineItemEditor.tsx — table + add row + delete row
  • IN TableFeed or dedicated query for labor/material/equipment pick lists with search
  • IN Client calls resolveEstimateLineItemUnitCost on selection to seed unit cost and unit label
  • IN Wire create/edit/delete to GraphQL from IE-1
  • IN Integration gate: show editor when estimate creationType = Itemized

Out of Scope

  • OUT Record-level overhead rollups (IE-3)
  • OUT PDF / invoice (IE-4, IE-5)

6. Acceptance Criteria ▼

  • User can add, edit, remove rows without full page reload
  • Selecting a catalog row fills unit cost via server resolve
  • Custom row allows manual description and costs with null catalog FKs
  • Per-line markup updates price column
  • Permissions match existing estimate edit rules
