Extend EstimateChangeOrderRecordCreationType with Itemized. Roll up EstimateLineItem costs and prices into EstimateChangeOrderTotals; keep record-level overhead and estimate markup in sync with line-derived subtotals. Optional contingency fields (reuse or add nullable decimals on ECO record).

1. Business Case ▼

Bid and job value must reconcile with the sum of line prices plus overhead, mirroring how the Full worksheet uses COGS subtotals.

2. Scope ▼

In Scope

  • IN Prisma enum extension Itemized on EstimateChangeOrderRecordCreationType
  • IN EstimateChangeOrderTotals.tsx branch for Itemized: line sum, overhead, markup, grand total, gross margin %
  • IN upsertFullEstimateChangeOrderRecord accepts line item payload; transactional save
  • IN Optional contingencyPercent / contingencyDollarValue on record if not already modeled

Out of Scope

  • OUT PDF layout (IE-4)
  • OUT Invoice mutation (IE-5)

4. Data Model ▼

enum EstimateChangeOrderRecordCreationType {
  BidFirst
  Full
  Itemized
}

Optional nullable contingencyPercent / contingencyDollarValue on EstimateChangeOrderRecord if not present; otherwise fold contingency into existing memo fields until schema approved.

5. API Surface ▼

input UpsertItemizedEstimateInput {
  estimateChangeOrderRecordId: ID!
  creationType: EstimateChangeOrderRecordCreationType!
  lineItems: [UpsertEstimateLineItemInput!]!
  overheadMarkup: Float
  estimateMarkup: Float
  bid: Float
}

Either extend upsertFullEstimateChangeOrderRecord with optional lineItems or add dedicated upsertItemizedEstimate mutation (implementation plan picks one).

  • MOD apps/api/prisma/schema.prisma
  • MOD EstimateChangeOrderRecord.ts (model + GraphQL)
  • MOD EstimateChangeOrderTotals.tsx

7. Acceptance Criteria ▼

  • Itemized totals panel shows cost subtotal, markup total, price subtotal, overhead, grand total
  • Saving estimate persists line items and header fields atomically
  • Switching creation type away from Itemized hides line editor with confirm if rows exist
  • Gross margin % matches defined formula
